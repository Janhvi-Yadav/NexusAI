# NexusAI — Python + React 

A full-stack AI chatbot built with:
- **FastAPI** (Python) backend
- **React + Vite** frontend
- Chat history saved in browser
- 4 modes: Chat, Study, Research, Docs
- PDF/TXT upload & Q&A
- Image input (vision)
- Voice input

---
